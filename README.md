# Vinherb-Skincare-Product-Website
A frontend cosmetic e-commerce website built with HTML, CSS, and Bootstrap 5.

## Features
- Home page
- Product listing page
- Product details page
- Shopping cart page
- Contact page
- Login / Register UI

## Technologies Used
- HTML5
- CSS3
- Bootstrap 5

## Description
This project focuses on building a responsive e-commerce interface with a clean UI layout and structured components.
